# dotfiles

## Install

```sh
make install
```

バックアップ作成が不要なとき

```sh
NO_BACKUP=1 make install
```

## Features

- よく使うコマンドエイリアスを設定
- path を設定 (not yest)
- prompt を設定
  - exit status を色付きで表示
  - カレントディレクトリ名を表示
  - git のブランチ名を表示
  - プロンプトの色を設定
- よく使うコマンドをインストール (not yet)
